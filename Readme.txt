Cross Lingual Document Translator Statistical Machine Translation Model

Cross Lingual Document Translator based on the IBM Model 1 and EM algorithm to translate text documents from english to french and vice versa, compare the generated translation against an accurate translation using Cosine Similarity and Jaccard Coefficient.

Requirements:
64-bit binary installation of Python 3.5+
16 gb of RAM

Files inlcuded:
training.py
clir.py
Documentation.docx
README.md

Compiling and Running the Code:
Open the clir.py using IDLE (Python 3.5). Press F5 to run the code. The program prompts the user to either train the model or enter a file path to translate text documents.
